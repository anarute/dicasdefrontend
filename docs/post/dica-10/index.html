Úteis e legais

Web Starter Kit [en] Um boilerplate¹ do Google para desenvolvimento web com foco em sites multi-dispositivos. Além de apresentar uma ótima documentação para boas práticas de desenvolvimento. ¹ No contexto web, boilerplate é como um esqueleto genérico de sites que já vem com várias ferramentas e configurações úteis para poupar seu tempo ao iniciar um projeto.

Animate.css [en] Um conjunto de animações em CSS prontas para você utilizar nos seus sites.

Os Simpsons em CSS puro [en] Ótimo para experimentar e brincar de fazer formas apenas com CSS e HTML. E de quebra aprender um pouco sobre animações em CSS (via Padovan)
