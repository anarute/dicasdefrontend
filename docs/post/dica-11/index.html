Novidades

Novo design do site “Can I Use” O caniuse é um site bastante útil para verificar o suporte dos navegadores desktop e mobile para os principais elementos front-end (como propriedades CSS3, elementos HTML5, APIs JavaScript, etc.) O redesign traz um visual mais clean e uma experiência de uso mais agradável, além da correção de alguns bugs e maior suporte para mobile. A nova versão ainda está em beta passando por testes e recebendo feedbacks antes de tomar o lugar da atual.

Artigos

Sobre desenvolvimento multi-device [6 min] Uma forma de classificar melhor os dispositivos para se poder proporcionar uma experiência de uso ideal em qualquer situação.

Úteis

CSS Shrink [en] Ferramenta para “encolher” seu CSS. Parece ser uma boa solução para projetos grandes em que uma refatoração pode custar muito caro.

Wikipedia minimalista [en] Uma proposta de design minimalista para a Wikipedia que otimiza a leitura de artigos.
